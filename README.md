# sede-foundations

**SEDE I: Foundations of Structural Entropy Dark Energy**

Paper I of the SEDE series: foundations of the structure-gated, horizon-entropy dark energy model.

Part of the USC research program (SEDE / ECCG / APDM unification).

## Status
Pre-submission working repository, seeded 2026-07-17 from a private research hub;
full research history (including audit trails and
retractions) lives there.

## Layout
- `paper/` — manuscript sources (md + tex + build script; built PDF included); `paper/tools/` holds the build machinery
- `src/` — the importable `sede` package plus `src/experiments/` (numbered numerical experiments) and `src/scripts/`
- `figures/` — figure generation; `results/` — generated figure outputs
- `reproduce.py` — single entry point that reproduces the results and figures
